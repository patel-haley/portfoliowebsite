Projects

Skycraft

2024

Skycraft is a prototype designed to assist first responders by providing 3D renders of disaster-affected areas using drone-captured video footage. The project integrates drone video capture, Raspberry Pi for temperature sensor data, and Google Cloud for video storage. On the backend, COLMAP and a 3D Gaussian Splatting model are used for 3D reconstruction. The frontend is built with React and Vite to display 3D models and temperature data in real time.

Technologies Used: React, Vite, Typescript, Python, OpenCV, COLMAP, Raspberry Pi
